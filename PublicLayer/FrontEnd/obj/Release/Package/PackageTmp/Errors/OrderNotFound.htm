Critical Error: Requested Order is not found

The system can not locate the Order.


Resolution:
This error occurs if the data store did not return the requested Order.

This can happen in the following situations:

1. The specified Order does not exist in the database.
2: The Order exists but you do not have enough rights to access it.
Make sure you are logged in.
