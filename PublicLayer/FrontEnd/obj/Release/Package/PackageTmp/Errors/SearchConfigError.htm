Critical Error: Search Exception

Search was not configured or misconfigured.


Resolution:
This error occurs typically when search index folders has not been configured or when directories where index files are stored are inaccessible. It can also be due to wrong paths specified in the configuration file. For more details on how to configure search please follow instructions outlined in our documentation.
